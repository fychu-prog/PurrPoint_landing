## 三、目前已知限制

購買前請確認以下限制是否符合你的使用需求。建議先完整使用 14 天試用版再決定購買。

### 3.1 語言支援
- **主要優化**：繁體中文、英文
- **部分支援**：簡體中文、日文等其他語系（準確度較低）
- 若文件以非優化語系為主，建議先用試用版測試

### 3.2 圖表（長條圖、圓餅圖等）
- 圖表以**圖片形式**保留在輸出的 PPTX 中，位置與原始 PDF 對應
- **無法點入修改圖表數據**
- 圖表的文字標籤（標題、說明文字）可正常編輯
- 如需修改圖表數字，建議在 PowerPoint 重新製作

### 3.3 項目符號與清單
- 文字內容**完整保留**
- 輸出為文字方塊，**非 PowerPoint 原生清單排版**
- 黑點（•）、數字（1. 2. 3.）為文字的一部分，無法一鍵套用清單樣式
- 如需更改符號格式，需手動調整

### 3.4 字體
- OCR 辨識只還原文字內容與座標，**無法識別原始字體**
- 輸出統一使用 macOS 系統字體
- 若原 PDF 使用特殊窄體字，同樣文字在系統字體下可能較寬，部分文字方塊需要手動微調寬度

### 3.5 平台
- 目前**僅支援 macOS**（Apple Vision Framework 限定）
- Windows 版本評估中，尚無確定時程

### 3.6 試用限制
- 免費試用版：每次轉換上限 **10 頁**
- 正式版（購買後）：無頁數限制

## 四、安裝說明：Gatekeeper 警告

由於 PurrPoint 目前尚未申請 Apple 開發者憑證，首次開啟時 macOS 會顯示「無法驗證開發者」警告。

**解決方法：**
1. 在 Finder 找到 PurrPoint.app
2. 按住 Control 鍵（或右鍵）點選 → 選「打開」
3. 在彈出視窗再選「打開」
4. 之後正常雙擊即可開啟，不會再出現警告

軟體完全本機運作，無網路存取，無法讀取你電腦上的其他檔案。
